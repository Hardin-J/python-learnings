## ğŸ› Troubleshooting

### **Common Issues:**

**1. "ModuleNotFoundError" when running a file**
- Make sure you're in the correct directory
- Use absolute paths or navigate to the file's directory

**2. "Invalid syntax" error**
- Check for missing colons (`:`) at the end of if/for/while/def statements
- Check indentation (Python uses 4 spaces)
- Check for matching parentheses and quotes

**3. "input() seems to hang"**
- The program is waiting for your input
- Type something and press Enter

**4. File not found errors**
- Check the file path
- Make sure you're running from the correct directory
